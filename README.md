# Agnostic Zoom
<br><br>A tool for zooming into arbitrary application windows and displaying them as an overlay in real-time.<br><br><br>
![alt text](https://i.imgur.com/FdIECHk.png)
<br><br><br>Window, zoom type, and hotkey selection. Easy integration to OBS, Streamlabs etc.<br><br><br>
![alt text](https://i.imgur.com/A1DjtSW.png)
<br><br><br>1/2 Zoom selection. Hotkey press displays the chosen screen zoomed into the middle region (half in each dimension of the screen). The source screen can still be controlled in real time. <br><br><br>
